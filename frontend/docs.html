📖 Sobre o projeto

Memórias com Amor é uma aplicação web para guardar e organizar memórias especiais com fotos e álbuns personalizados. O projeto foi desenvolvido com foco em simplicidade, beleza e facilidade de uso.

Status: Em produção Versão 2.0.0

🛠️ Tecnologias utilizadas

FastAPI PostgreSQL SQLAlchemy JWT Docker Cloudinary HTML5 CSS3 JavaScript Vercel Render

✨ Funcionalidades

  • 🔐 Autenticação — Registro e login com JWT (tokens de acesso e refresh)
  • 📚 Álbuns — Crie, edite e delete álbuns com categorias (Amor, Viagem, Aniversário, etc.)
  • 📸 Fotos — Upload de imagens com otimização automática (Cloudinary)
  • ❤️ Favoritos — Marque suas fotos preferidas
  • 📅 Linha do Tempo — Visualize todas as fotos organizadas por data
  • 👤 Perfil — Edite nome, bio e foto de perfil
  • 📱 Responsivo — Funciona em desktop, tablet e mobile
  • 🔒 Segurança — Rate limiting, headers de segurança e validação de senha

📡 Rotas da API

Todos os endpoints estão disponíveis em: https://memorias-com-amor.onrender.com/api

🔐 Autenticação

  • POST /auth/register — Registrar novo usuário 5/min
  • POST /auth/login — Login 10/min
  • POST /auth/refresh — Renovar token de acesso
  • GET /auth/me — Dados do usuário logado

📚 Álbuns

  • GET /albums — Listar todos os álbuns
  • POST /albums — Criar um novo álbum
  • GET /albums/{id} — Buscar um álbum
  • PATCH /albums/{id} — Atualizar um álbum
  • DELETE /albums/{id} — Deletar um álbum

📸 Fotos

  • POST /photos/upload — Upload de uma foto
  • GET /photos/album/{id} — Listar fotos de um álbum
  • GET /photos/all — Listar todas as fotos do usuário
  • GET /photos/{id} — Buscar uma foto
  • PATCH /photos/{id} — Atualizar metadados da foto
  • DELETE /photos/{id} — Deletar uma foto
  • POST /photos/{id}/fav — Favoritar/desfavoritar

👤 Perfil

  • GET /profile/me — Obter perfil do usuário
  • PATCH /profile/me — Atualizar perfil (nome, bio)
  • POST /profile/upload-pic — Upload de foto de perfil

📊 Saúde

  • GET /health — Verificar status da API

🚀 Como usar

  1. Registre uma conta — Clique em "Criar conta" no frontend
  2. Faça login — Entre com seu e-mail e senha
  3. Crie um álbum — Clique em "Novo álbum" e preencha os dados
  4. Adicione fotos — Dentro do álbum, clique no botão de upload
  5. Explore — Favoritos, linha do tempo e perfil
